# 量子坍缩选号 · Quantum SuperEnalotto

用真正的量子随机性为意大利大乐透(SuperEnalotto)选号。每次点击「观测」,应用会向澳大利亚国立大学(ANU)的量子随机数源请求一批数字——它们来自对真空电磁场零点涨落的实时测量。根据量子力学,这些数字在测量前并不存在确定的值:不是我们不知道,而是宇宙自己也"没决定"。

> 道生一,一生二,二生三,三生万物。——《道德经》第四十二章

## 特性

- ⚛ 熵源优先使用 [ANU QRNG](https://qrng.anu.edu.au/)(量子真空涨落),不可达时自动回退到 `crypto.getRandomValues`,界面如实标注当前熵源
- 拒绝采样消除取模偏差,保证 1–90 严格等概率
- 按 SuperEnalotto 规则:6 个主号(1–90 不重复)+ Jolly(剩余 84 个中抽取)+ SuperStar(独立抽取)
- 波函数坍缩动画:90 个数字的叠加态逐个退相干
- 支持 `prefers-reduced-motion`

## 运行

```bash
npm install
npm run dev
```

构建生产版本:

```bash
npm run build
```

## 诚实声明

量子力学保证真随机**不可预测**——包括对这个应用自己。本应用产生的号码与任何选号方式的中奖概率完全相同(头奖约 1/622,614,630)。它不会提高你的胜率,只是让你的号码诞生于一次真实的波函数坍缩。仅供娱乐,理性购彩。
